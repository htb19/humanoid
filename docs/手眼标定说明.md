# 一、手眼标定说明

## 1.安装

```bash
sudo apt install ros-humble-aruco-ros
```


## 2.编译

```bash
colcon build --packages-select camera_calibration robot_simulation
source install/setup.bash
```


## 3.启动

```bash
# 仿真环境下
ros2 launch robot_simulation calibration.launch.py
```


## 4.采集数据并计算位姿

```bash
# 保存数据：（至少保存三组）
ros2 service call /save_handeye_pair std_srvs/srv/Trigger

# 计算并发布结果：
ros2 service call /compute_handeye std_srvs/srv/Trigger
```
