# 一、伺服环境配置

## 1.安装moveit servo

```bash
sudo apt install ros-humble-moveit-servo
```


# 二、编译启动

```bash
# 编译：
colcon build --packages-select robot_simulation servo_control
source install/setup.bash

# 启动：
# 仿真环境（含gazebo）
ros2 launch robot_simulation visual_servoing.launch.py

# 仿真环境（仅rviz）
ros2 launch robot_commander robot_moveit.launch.xml
ros2 launch servo_control servo.launch.py
```

# 三、使用说明

```bash
# 激活伺服节点
ros2 service call /servo_node/start_servo std_srvs/srv/Trigger {}

# 指令发布测试（另一个终端）

  # 含gazebo
ros2 run servo_control test_servo_cmd --ros-args -p use_sim_time:=true

  # 仅rviz
ros2 run servo_control test_servo_cmd

# 现象：机器人左臂缓慢抬起
```
